🎨 COLOR ZONES QUICK REFERENCE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🔵 BLUE - Cool
   • < 75°F (< 24°C)
   • Status: All good!
   • Action: None needed

🟡 YELLOW - Mild
   • 75-84°F (24-29°C)
   • Status: Normal operation
   • Action: Monitor if sustained

🟠 ORANGE - Warm
   • 85-94°F (29-34°C)
   • Status: Heavy workload
   • Action: Consider cooling measures

🔴 RED - Hot
   • ≥ 95°F (≥ 35°C)
   • Status: Very hot!
   • Action: Reduce load, improve cooling

🚀 HOW TO VIEW THE DEMOS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Terminal Preview:
  ./color_preview.sh

Documentation:
  open COLOR_DEMO.md
  open VISUAL_GUIDE.md

Live App Testing:
  1. Launch MenuTemp
  2. Note current color
  3. Open intensive app (video export, game, etc.)
  4. Watch color change in real-time
  5. Close apps and watch it cool down

💡 USAGE IN MENUTEMP APP
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Menu Bar Display:
  🌡️  XX.X°F  ← This text changes color

Details Dialog:
  Click menu → Details (⌘D)
  Shows current zone with indicator

Toggle Units:
  Click menu → Celsius/Fahrenheit (⌘U)
  Colors adapt to °C thresholds

🎬 CREATING SCREENSHOTS

To capture actual MenuTemp screenshots:
